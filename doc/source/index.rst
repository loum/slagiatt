###############################################################
Web API Micro-services: Seemed Like a Good Idea at the Time ...
###############################################################

SLAGIATT aims to be a base for implementing the Microservices
Architectural pattern.  A good description of Microservices can
be found `here <http://martinfowler.com/articles/microservices.html>`_.

As a Microservice, SLAGIATT is based on the Python Flask
mirco-webframework and SQLAlchemy as the DBA abstraction layer.

SLAGIATT currently bundles with a functioning service that demonstrates
this pattern.
